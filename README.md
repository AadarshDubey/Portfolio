A premium, production-ready portfolio website showcasing expertise in Generative AI, Full-Stack Development, and Cinematic Web Experiences.

## � Overview

This portfolio is designed to deliver a high-end, immersive experience that bridges the gap between complex AI engineering and premium digital aesthetics. It features:

- **Cinematic Hero Animation**: A scroll-driven sequence using 192 sequential frames rendered via high-performance HTML5 `<canvas>`.
- **Dynamic Projects Showcase**: Interactive demos highlighting real-world AI implementations, RAG pipelines, and agentic workflows.
- **Modern Tech Stack**: Built with a focus on performance, accessibility, and state-of-the-art UI/UX.

## 🛠️ Tech Stack

- **Core**: Next.js 16 (App Router), TypeScript
- **Styling**: Tailwind CSS v4, Framer Motion
- **Icons & Assets**: Lucide Icons, Custom Sequential Frames
- **Optimization**: SEO-ready with dynamic sitemaps, JSON-LD, and high-performance asset loading.

## 🚀 Key Features

### 🎬 Scroll-Driven Cinematic Engine
The hero section utilizes a custom-built frame sequence engine that synchronizes high-resolution imagery with the user's scroll position, creating a professional parallax effect that responds to every micro-interaction.

### 🤖 Generative AI Focus
Built specifically for a GenAI Engineer, the site structure emphasizes technical proficiency in LLM orchestration, vector databases, and agentic systems through bespoke demo components for every skill.
